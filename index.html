Minesweeper

Minesweeper is refreshed version of popular game, who is well-known by literally everybody.

It's my second desktop application (and most advanced) project created in Java using JavaFX Framework.

The rules are same like in typical Minesweeper game, which can be found on every Windows operating system. In my version, default number of mines is ten, but it can be changed easily in code.

Player can press the tiles with left mousebutton to reveal the number (representing mines around) or blank space.

With using right mausebutton, player can flag possible mine.
If you press mine tile - you immediately lose.
If you flag all mines correctly and there are no more possible tiles to reveal - you win!

Hangman

Everybody knows paper and pencil game named Hangman.
It's my first serious Java command line application.

It works as follows:

One player gives computer the password, it is beeing encrypted and showed in the form of dashes on the console.


Player two has to guess all the letters.
Every mistake adds part of hangmans body on gallow.

If he is hanging with all his body - you loose.


If you guess all the letters- you win!

This project was good opportunity for me to learn not only object programming principles, but
unit testing using JUnit Framework as well.
